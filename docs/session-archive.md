# docs/session-archive.md: full session history archive

`HANDOFF.md` keeps only a short, rolling summary of recent sessions so it stays small
and fast to read at the start of every session. The full detailed history lives here,
oldest first. When `HANDOFF.md`'s "Session log" section grows past roughly 5 entries,
move the oldest entries here verbatim (do not summarise further) and leave a one-line
pointer in `HANDOFF.md`.

This file is reference only: nothing here is "live" and nothing needs to stay in sync
with it. It exists so history is never lost.
