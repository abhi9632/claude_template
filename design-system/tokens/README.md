# design-system/tokens/

Reference copies of original design-system token values (for example, exported from a
design handoff bundle or a Figma tokens plugin), kept here for context only. The
canonical, machine-consumed tokens live in this project's actual stylesheet or config
(see `design-system/README.md` > Visual foundations for where that is).

Do not import files from this folder directly into the app. Use them as a reference
when porting values into the canonical token file, and document any naming differences
in `design-system/README.md` > Token mapping.

This folder can stay empty until a design system is adopted.
